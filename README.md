# Afrisoft AS website

GitHub Pages-ready website for Afrisoft AS and **Legacy of the Sphinx**.

## Correct structure

```text
index.html
games.html
privacy.html
styles.css
script.js
assets/
  header.jpg
  legacy-01.jpg
  legacy-02.jpg
  legacy-03.jpg
  legacy-04.jpg
  legacy-05.jpg
```

## Publish on GitHub Pages

1. Upload all files and the `assets` folder to the root of the repository.
2. Open **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose your main branch and **/(root)**.
5. Save and wait for the published URL.

Use the final `privacy.html` URL in Google Play Console.
