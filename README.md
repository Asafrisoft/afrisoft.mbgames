# Afrisoft AS website

Modern static website for Afrisoft AS and the mobile game **Legacy of the Sphinx**.

## Files

- `index.html` — company homepage
- `games.html` — dedicated game page
- `privacy.html` — Privacy Policy for Legacy of the Sphinx
- `styles.css` — complete responsive design
- `assets/` — game artwork

## Publish with GitHub Pages

1. Create a public repository.
2. Upload all files and the `assets` folder.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder.
6. Save.

Your Privacy Policy URL will end in `/privacy.html`.
